   Đăng nhập và phân quyền đối với tài khoản:
   Quy trình:
   - Người sdung phải cung cấp thông tin cho hệ thống để hệ thống kiểm tra xem có quyền truy cập vaò hệ thống hay không 
   - Hệ thống kiểm tra thông tin đăng nhập, nếu hợp lệ hệ thống sẽ cấp giấy chứng nhận (principal) cho phía client dưới dạng 1 cookie
   - Phía client phải lưu giữ cookie và xuất trình cookie mỗi khi thực hiện các yêu cầu lên web server (nằm trong phần header của request )
   - Hệ thống sẽ căn cứ vào giấy chứng nhận để kiểm tra tính hợp lệ của người dùng 
   2 Thuật ngữ:
   - Athentication
   - Authorization
   Trong ASP.NET Core muốn dùng chức năng đăng nhập/phân quyền thì phải đăng ký
   Đối với các Controller hoặc Action mà yêu cầu phải đăng nhập mới được sử dụng, đăjt chỉ thị [Authorize] ở trước
   Sử dụng chỉ thị [AllowAnonymous] trước Action mà không yêu cầu phải đăng nhập để sử dụng
   Trong trường hợp cần phải kiểm tra "quyền" đối với Controller hoặc Action 
      [Authorize(Roles = "Danh_sách_quyền")]
   Trong đó: Danh sách quyền là tên của các Role, viết phân cách nhau bới dấu phẩy
   Trong Action hoặc View, muốn lấy giấy chưnngs nhận (ClaimsPrincipal), thông tin thuộc tính user
      WebUserData -> CreatePrincipal -> ClaimsPrincipal

@section Scripts {
    <script>
        function scrollToProducts() {
            const target = document.getElementById('product-list-target');
            if (target) target.scrollIntoView({ behavior: 'smooth' });
        }

        function doSearch(page) {
            var form = $("#formSearch");
            var url = form.attr("action");
            var postData = form.serializeArray();
            postData.push({ name: "Page", value: page });

            $.ajax({
                url: url,
                type: "GET",
                data: postData,
                error: function () {
                    alert("Yêu cầu của bạn không thể hoàn thành.");
                },
                success: function (data) {
                    $("#searchResult").html(data);
                    fixImages();
                }
            });
        }

        function fixImages() {
            const defaultSrc = "@defaultImg";
            const images = document.querySelectorAll('.product-img-container img, .product-item img');
            images.forEach(img => {
                if (!img.getAttribute('data-error-handled')) {
                    img.addEventListener('error', function() {
                        this.src = defaultSrc;
                        this.style.padding = "20px";
                        this.style.backgroundColor = "#fff";
                        this.style.objectFit = "contain";
                    });
                    img.setAttribute('data-error-handled', 'true');
                }
            });
        }

        $(document).ready(function () {
            // Tự động load trang đầu tiên sử dụng page từ model hoặc mặc định
            doSearch(@Model.Page);

            $("#formSearch").submit(function (e) {
                e.preventDefault();
                doSearch(1);
            });
        });
    </script>
}

@section Scripts {
    <script>
        $(document).ready(function () {
            // Lắng nghe sự kiện click vào nút "Thêm giỏ"
            $('.btn-add-to-cart').click(function (e) {
                e.preventDefault();

                let btn = $(this);
                let productId = btn.data('id');

                // Tìm ảnh sản phẩm nằm trong cùng khung 'product-item' với nút vừa bấm
                let imgToFly = btn.closest('.product-item').find('.product-img');

                // Gửi yêu cầu Ajax lên Controller
                $.ajax({
                    url: '/Cart/AddToCart',
                    type: 'POST',
                    data: { id: productId, quantity: 1 },
                    success: function (response) {
                        if (response.success) {
                            // 1. Kích hoạt hiệu ứng bay (hàm này nằm trong _Layout)
                            if (typeof doFlyToCart === "function") {
                                doFlyToCart(imgToFly);
                            }

                            // 2. Cập nhật số lượng trên Badge giỏ hàng
                            // Đợi 0.8 giây cho hộp quà bay gần tới rồi mới đổi số
                            setTimeout(function () {
                                $('#cart-count').text(response.totalItems);
                            }, 800);
                        } else {
                            alert(response.message || "Không thể thêm sản phẩm vào giỏ hàng.");
                        }
                    },
                    error: function () {
                        alert("Lỗi kết nối đến hệ thống.");
                    }
                });
            });
        });
    </script>
}
